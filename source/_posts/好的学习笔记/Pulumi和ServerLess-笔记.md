# ServerLess

* 服务器资源可以动态生成，它提供了 api 进行生成。
* 比如文件上传流量是不确定的，我们没有必要购买一台长久的服务器，因为大多数时候我们用不了那么多资源。这时候你可以使用这个技术，将文件服务接口连接服务器 api，实现动态的资源分配。
* 同时，云服务的服务器路由可能很长，而且接口封装不符合我们的使用场景，我们可以封装自己的 CDN，一方面有更实用的 api，另一方面可以自己流量管理。
